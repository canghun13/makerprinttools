# MakerPrintTools

Practical 3D printing calculators for personal makers and small print operations.

- Production: https://makerprinttools.com
- Stack: static HTML, CSS, and vanilla JavaScript
- Project continuity and rules: see `handover.md`

## Local checks

With Node.js installed:

```text
node scripts/generate-calculator-pages.mjs
node scripts/qa.mjs
```

Serve the repository root with any static HTTP server to test locally. No API, database, build step, or package installation is required for production.
