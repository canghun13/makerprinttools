How the calculation works

Total = (material + electricity + labor + machine) ÷ (1 − failure rate). Currency is intentionally neutral: keep every price input in the same currency.

How to use this result

A cost is not automatically a selling price. Add tax, platform fees, packaging, design time, and profit separately when they apply. Measure average power with a plug-in meter for better accuracy.

Example

A 120 g, eight-hour print from a 1 kg spool can have modest material and power costs, while labor and machine allowance dominate the true job cost.
