Infill Material Estimator

Estimate internal material volume and mass from a model envelope, shell allowance, infill percentage, and material density.

Inputs

Result

—

How the calculation works

Infill material = (model envelope − shell volume) × infill percentage × density.

How to use this result

Use it for early design comparison when a slicer estimate is not available. It is deliberately an approximation: real infill geometry, top layers, supports, and voids change the final mass.
