How the calculation works

Material cost = filament used × (1 + waste allowance) ÷ spool net weight × spool price.

How to use this result

This is intentionally material-only. Use the 3D Print Cost Calculator when you also need electricity, labor, machine allowance, and expected failed-print costs.

Example

A 120 g print from a 1 kg spool priced at 22, with 3% allowance, uses 123.6 g of billable material.
